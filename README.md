# DLL说明
android_zhizi.dll 是一个 通过adb获取android基本信息的dll，最初开发目的是为了适配易语言。

# 开发环境
IDE: Visual Sutdio 2022
C++: vc2022

# 生成方法
Win32平台(比如易语言)使用,请选择Relese X86生成方法。
Win64平台使用,推荐选择Relese X64生成方法,且Relese X86生成方法可用。
